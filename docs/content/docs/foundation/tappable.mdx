## Custom Bounce Animation

```dart copy
// Default animation.
FTappableMotion(bounceTween: FTappableMotion.defaultBounceTween);

// No animation.
FTappableMotion(bounceTween: FTappableMotion.noBounceTween);

// Custom tween.
FTappableMotion(bounceTween: Tween(begin: 1.0, end: 0.97));
```
