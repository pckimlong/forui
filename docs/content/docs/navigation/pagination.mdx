## CLI

To generate and customize this style:

```shell copy
dart run forui style create pagination
```

## Usage

### `FPagination(...)`

```dart copy
FPagination(
  control: FPaginationControl.managed(
    pages: 20,
    initial: 4,
    showEdges: false,
    siblings: 2,
    onChange: (page) {},
  ),
  style: (style) => style.copyWith(...),
  previous: const Placeholder(),
  next: const Placeholder(),
);
```
